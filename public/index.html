Supported formats

CSV, TSV, JSON array, or a single JSON object.

Imported leads do not publish directly. They become Radar candidates with confidence, review, and risk scoring first.

CSV headers can use name, deadline, websiteUrl, submissionUrl, contentTypes, formats, prizeMoney, entryFee, organizerName, venueName, rulesPage, transparentFees, prizeDetails. JSON can use the same fields as Scout leads, including evidence and warningSignals. Bad rows are skipped with row-level errors so one messy lead does not block the whole import.
